```ts
const reverseInteger = (myInteger) => {
   const reversed =  parseInt(myInteger.toString().split('').reverse().join(''))
   return Math.sign(myInteger) * reversed
}
```

### Explanation
The function takes an integer as an argument and converts it to a string.
It then splits the string into an array of characters, reverses the array, 
and joins it back to a string. The string is then converted back to an integer. 
The function then multiplies the integer by the sign of the original integer to ensure that the result has the same sign as the original integer.

## Complexity Analysis
The time complexity for this function is O(n) where n is the number of digits in the integer.
